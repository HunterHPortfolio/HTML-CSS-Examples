Upgrade RAM
Just by upgrading the RAM in a computer, you can improve the performance dramatically. If you are running Windows XP it is recommended that you have AT LEAST 1GB of memory installed. If you are running Windows 7, you should have AT LEAST 2GB's of memory, even though the minimum requirement is 1GB. The more RAM installed in your computer the better. When you are running a 32 bit operating system, you can have up to 4GB of RAM, when running a 64 bit system, you can have more than that.

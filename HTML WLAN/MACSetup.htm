MAC Setup

Steps for Setting up the MAC TS (Testing Station) and Software Usage

All of the MAC testing uses the TS with the DK Atheros Card hardware installed, as well as the Sniffer TS equiped with a packet sniffer.
In order to set up MAC testing - follow these steps:

    1: On the MAC TS, navigate to C:\Program Files\DK and locate MAC.txt
    2: Comment out the previous device and enter the new device being tested with the same information as the device that was commented out.
    3: Configure the MAC Address and IP -> SSID should be "MAC"
    4: Open the Command prompt and type this command: "CD C:\Program Files\DK>mdk" (Don't Press Enter)
    5: Open the Folder on desktop that has all the current MAC Scripts and find the test you are conducting (labeled appropriately)
    6: Drag the script file into the command prompt (it should have mdk now followed by script location)
    7: Open Omnipeek in order to capture all of the MAC frames
    8: Start OmniPeek capture followed by pressing enter in the command window
    9: Hit Ctrl-S on OmniPeek, save the results.
    10: Make sure to run each test twice! Have fun!

Settings to Change While Running a MAC Base Test

    Test 1.1.3: Continuously Ping the TS (from the Sniffer, Run the Command Prompt, Enter "ping 192.168.1.101 -t")
    Some tests you might have to start first then ping.
    Test 1.1.5: Continuously Ping the TS
    Test 1.1.6: Continuously Ping the TS
    Test 1.1.9: Set Fragmentation to 256 (Could be complicated but if it's not configurable via GUI),
    an example command if you are using the terminal to change a devices settings is to enter this command "iwconfig ath0 frag 256"
    Test 1.1.12: Continuously Ping the TS
    Test 1.2.1a: Set Fragmentation to 2346 (or off) and RTS to 512 "iwconfig ath0 rts 256"
    Test 1.2.1b: Set Fragmentation to 512 and RTS to 512
    Test 1.2.2a: Set Fragmentation to 512 and RTS to 256
    Test 1.2.2b: Set Fragmentation to off and RTS to 1024
    Test 1.2.3: Set Fragmentation to 512 and RTS to 512
    Test 1.2.4a: Set Fragmentation to 512 and RTS to 600
    Test 1.2.4b: Set Fragmentation to 301 and RTS to off
    Test 1.2.5a: Set Fragmentation to off and RTS to 256
    Test 1.2.6: Set Fragmentation to off and RTS to 512
